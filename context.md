# Context: Trener Biegania Landing Page

## Cel Projektu
Stworzenie nowoczesnego, szybkiego i responsywnego Landing Page dla trenera biegania (wymyślone dane) pozycjonowanego pod rynek lokalny: Straszyn, Pruszcz Gdański, Trójmiasto. 

## Stack Technologiczny
- **Frontend**: Single-file HTML5 + Tailwind CSS (skrajnie wysoka wydajność)
- **Stylistyka**: Jasna, ciepła kolorystyka premium (kremowe tło #FAF8F5, zielone, limonkowe i pomarańczowe akcenty energetyczne ze Stitch).
- **Zasoby**: Unsplash (zdjęcia biegowe), FontAwesome (ikony).
- **Hosting**: Docelowo Vercel lub Netlify (darmowy statyczny hosting).

## Struktura Strony (UX-First)
1. **Sticky Header & Navigation** (Logo, O mnie, Opinie, Plany, Kontakt, Sociale, Mobile Hamburger Menu).
2. **Hero Section** (Chwytliwy nagłówek SEO lokalny, CTA "Rozpocznij Trening", social proof bar z metrykami).
3. **Sekcja "O mnie"** (Bio trenera, sukcesy w biegach ogólnopolskich, certyfikaty w formie kart).
4. **Opinie (Testimonials)** (Social proof - 3 zadowolonych podopiecznych, cele sportowe).
5. **Cennik (Plany Treningowe)** (3 poziomy cenowe: Start, Pro [wyróżniony], Elite).
6. **Kontakt** (Przyciski do szybkiego dzwonienia/pisania oraz linki do social mediów).
